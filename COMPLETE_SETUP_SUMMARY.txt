================================================================================
❓ TROUBLESHOOTING
================================================================================

ISSUE: "Port 8000 already in use"
FIX:   Change PORT in .env file
       Or: kill the process using it

ISSUE: "Cannot connect to API"
FIX:   Check if service is running
       Check logs: docker-compose logs
       Verify .env configuration

ISSUE: "Agent takes too long"
FIX:   This is normal (5-30 seconds typical)
       Check execution_time in response
       May be OpenRouter API latency

ISSUE: "Memory errors"
FIX:   Increase memory in docker-compose.yml
       Or: Run on machine with more RAM

ISSUE: "SSL certificate errors"
FIX:   Check certificate paths in nginx.conf
       Verify Let's Encrypt renewal
       Check certificate expiration

See detailed troubleshooting in DEPLOYMENT_GUIDE.md
