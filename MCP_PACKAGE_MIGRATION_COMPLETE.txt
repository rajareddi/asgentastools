🚀 USAGE
═══════════════════════════════════════════════════════════════════════════════

Import from package:
  from mcp_package import (
      MCPConfiguration,
      MCPInstaller,
      mcp_manager,
      mcp_agent,
      mcp_agent_available
  )

Configure MCP:
  config = MCPConfiguration()
  config.enable_server("filesystem")
  config.enable_server("web")
  config.enable_all_servers()

Use MCP Agent:
  from agents import Runner
  result = await Runner.run(
      mcp_agent,
      input="Your task",
      max_turns=5
  )

Access via API:
  curl -X POST http://localhost:8000/run \
    -d '{"prompt":"task","agent_type":"mcp","max_turns":5}'
