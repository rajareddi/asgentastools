💻 QUICK START

### Step 1: Install MCP Dependencies

```bash
pip install -r requirements.txt

# Or specific servers
pip install mcp
pip install mcp-server-filesystem
pip install mcp-server-web
pip install mcp-server-git
```

### Step 2: Enable MCP in Your Code

```python
from mcp_package import MCPConfiguration

config = MCPConfiguration()
config.enable_server("filesystem")
config.enable_server("web")
config.enable_all_servers()  # Enable all

servers = config.get_enabled_servers()
```

### Step 3: Use MCP Agent

```python
from mcp_agent import mcp_agent
from agents import Runner

result = await Runner.run(
    mcp_agent,
    input="Read README.md and summarize it",
    max_turns=5
)
```

### Step 4: Access via API

```bash
# List agents (MCP now included)
curl http://localhost:8000/agents

# Run MCP agent
curl -X POST http://localhost:8000/run \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "List all Python files in current directory",
    "agent_type": "mcp",
    "max_turns": 3
  }'
```

🔧 CONFIGURATION

### Enable Specific Servers

```python
from mcp_package import MCPConfiguration

config = MCPConfiguration()

# Enable individual servers
config.enable_server("filesystem", root_dir="/safe/path")
config.enable_server("web")
config.enable_server("git")
config.enable_server("sql", database_url="sqlite:///db.sqlite")

# Get enabled servers
servers = config.get_enabled_servers()
```

### Start MCP Servers

```python
from mcp_integration import mcp_manager

# Register servers
mcp_manager.register_built_in_servers()

# Start servers
await mcp_manager.start_all_servers()

# Get status
status = mcp_manager.get_server_status()
print(status)
```
